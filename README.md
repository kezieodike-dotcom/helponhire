# Help On Hire

Help On Hire is a React/Vite frontend for an on-demand professional services and staffing marketplace. The app includes service discovery, a booking flow, business staffing requests, professional onboarding, contact forms, and a simulated WhatsApp-style support widget.

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Lucide React icons

## Getting Started

Prerequisite: install Node.js 18 or newer.

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the local development server:

   ```bash
   npm run dev
   ```

3. Open the URL Vite prints in your terminal. By default this is usually:

   ```text
   http://localhost:5173
   ```

If that port is already busy, Vite will automatically choose the next available port and print the correct URL.

## Available Scripts

- `npm run dev` starts the Vite development server.
- `npm run build` creates a production build in `dist`.
- `npm run preview` previews the production build locally.
- `npm run lint` runs the TypeScript type check.

## Notes

This repository is currently a frontend prototype. Form submissions, booking assignment, and chat responses are simulated in the browser and are not yet connected to a backend, database, payment provider, or notification service.
